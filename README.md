# crud-postgres

> A Vue.js project with express

## Requirements
* Node
* NPM or YARN

### Set Var Environment

1. Copy to `env.example` into `.env`

        cp env.example .env

1. Edit values in `.env`

        nano .env

### Install dependencies
``` bash
yarn install
```
### Build Static Setup

``` bash
# build for development
yarn run dev

# build for production with minification
yarn run build
```

### Build Server Setup
``` bash
# build for development
yarn run server:dev

# build for production with minification
yarn start
```
